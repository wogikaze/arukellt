## What is the Playground?

The Arukellt playground is an interactive browser-based environment for exploring
the Arukellt programming language. It runs entirely **client-side** — your code
never leaves your browser.

### v1 Capabilities

| Feature | Available | Notes |
|---------|-----------|-------|
| Code editing | ✅ | Syntax-highlighted editor |
| Parse | ✅ | Full parser via Wasm (`wasm32-unknown-unknown`) |
| Type checking | ✅ | Diagnostics displayed inline |
| Format | ✅ | Auto-format with the Arukellt formatter |
| Diagnostics | ✅ | Structured error/warning output with codes |
| Curated examples | ✅ | Pre-loaded example programs |
| Share / permalink | ✅ | Fragment-based URLs — no server needed |
| Full compilation | ❌ | v2+ (requires freestanding target) |
| Program execution | ❌ | v2+ (requires runtime support) |

### How It Works

The playground packages the Arukellt parser, type checker, formatter, and diagnostics
engine as a Wasm module compiled to `wasm32-unknown-unknown`. This module runs in a
Web Worker so the UI remains responsive. See [ADR-017](../adr/ADR-017-playground-execution-model.md)
for the full execution model decision.

### Getting Started

1. Open the playground in your browser
2. Type or paste Arukellt code in the editor
3. Diagnostics appear automatically as you type
4. Use the **Format** button to auto-format your code
5. Try a curated example from the **Examples** dropdown
6. Use **Share** to create a permalink for your code
