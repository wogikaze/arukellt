01 · Identity

なにものか

Rust 風の構文を持つ静的型付け言語。出力は Wasm（WASI）。ホスト I/O は std::host::* に隔離され、 コンパイラ自身も Arukellt で書かれた selfhost 実装が本体。

静的型付け Wasm-first / GC-native (T3) LLM-friendly (import/use 二層) Self-hosting compiler Component Model / WIT Trait-first stdlib 移行中

構文の味

fn / let / match / Result・Option / ?。struct・enum・ジェネリクス（制限付きモノモーフ化）。

メモリモデル (T3)

全値が Wasm GC ヒープ。線形メモリは I/O マーシャリング用。

ホスト境界

stdio / fs / env / process などは std::host::*（ADR-011）。純粋計算と I/O を分離。

正本の読み方

挙動 → docs/current-state.md。設計意図 → docs/adr/。作業キュー → issues/open/。

hello.ark

use std::host::stdio

fn main() {
    stdio::println("Hello, Arukellt!")
}

Result + match

fn divide(a: i32, b: i32) -> Result<i32, String> {
    if b == 0 {
        Err(String_from("division by zero"))
    } else {
        Ok(a / b)
    }
}

05 · Run & Verify

動かし方・検証

実行は wasmtime（≥ 30）上の selfhost wasm。解決順: $ARUKELLT_SELFHOST_WASM → .build/selfhost/arukellt-s3.wasm → bootstrap/arukellt-selfhost.wasm

実行・コンパイル

# 実行 scripts/run/arukellt-selfhost.sh run hello.ark # T3 向けコンパイル scripts/run/arukellt-selfhost.sh compile \ --target wasm32-wasi-p2 hello.ark # 型チェックのみ scripts/run/arukellt-selfhost.sh check hello.ark

07 · Stdlib

標準ライブラリの方針

ADR-036 / issue #709 — trait-first, type-first, method syntax へ移行中。 公開 API は impl Type とメソッドを優先。

Prefer

  • s.split(sep)
  • v.push(x)
  • n.to_string()
  • Vec::new()
  • String::from("x")

非推奨（公開面）

  • split(s, sep)
  • push(v, x)
  • i32_to_string(n)
  • Vec_new_i32()
  • String_from を主 API に

フリー関数・モノモーフヘルパーは内部・ブリッジ用として残る。ホスト API は std::host::stdio 等。
