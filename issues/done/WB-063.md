## Done When

- `.ar` files prefixed with `#!/usr/bin/env -S chef run` compile and run through `chef run` instead of failing on the launcher line
- a direct-execution regression proves `./file_name.ar` works when `chef` is available on `PATH`
- exactly one bundled example file demonstrates the shebang form and the example matrix stays green

## Notes

- verified with `cargo fmt`, `cargo test -p lang-core`, `cargo test -p arktfmt`, `cargo test -p chef`, `cargo test -p arktc --test examples`, `cargo test -p arktc --test workboard`, and `cargo test`
- `lang-core` now strips a leading shebang line before lexing/parsing, `arktfmt` preserves it while formatting the rest of the file, and `chef` has a direct-execution regression using a temporary `chef` symlink on `PATH`
- added `example/shebang_hello.ar` as the single bundled shebang example and registered it in `example/meta/matrix.json`
