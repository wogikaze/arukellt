# 807 — Fixture parity: 367 remaining failures

## Summary

The selfhost fixture parity harness (#287, done) reports 367 fixture
failures. #287 built the harness; this issue owns the remaining failures.

## Exact failure scope

367 fixtures fail when `python3 scripts/manager.py selfhost fixture-parity`
runs the selfhost compiler against the fixture suite. Each failure is a
fixture where the selfhost compiler produces incorrect output or crashes.

## Machine-readable baseline

The failing fixture IDs are captured in the verify full receipt
(`docs/data/verify-full-receipt.json`) under `checks[]` where
`check_id == "fixture_parity"` and `items[]` where `result == "fail"`.

The receipt is generated by `scripts/gen/write-verify-receipt.py` from
`python3 scripts/manager.py verify full` output.

## Owner

compiler/runtime team

## Removal condition

Each failing fixture is fixed when the selfhost compiler produces the
correct stdout for that fixture. The fixture is removed from the failing
set when `selfhost fixture-parity` reports it as passing.

## Validation command

```bash
python3 scripts/manager.py selfhost fixture-parity
```
