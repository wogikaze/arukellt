# 815 — Diagnostic/T3 compile skips

## Summary

49 fixtures are skipped in the diagnostic parity and T3 compile checks.
These fixtures have missing diagnostics or incomplete compile paths in
the selfhost compiler.

## Exact failure scope

- 23 T3 compile skips: fixtures listed in `scripts/check/check-t3-wasm-validate.py`
  `T3_COMPILE_SKIP` and fixtures that time out during compilation.
- 26 diagnostic parity skips: fixtures in `scripts/selfhost/checks.py`
  `DIAG_PARITY_SKIP` plus fixtures whose `.diag` golden file is not yet
  present.

## Machine-readable baseline

The skipped fixture IDs are captured in the verify full receipt
(`docs/data/verify-full-receipt.json`) under `checks[]` where
`check_id == "t3_wasm_validate"` or `check_id == "diag_parity"` and
`items[]` where `result == "skip"` and `owner_issue == "815"`.

The receipt is generated by `scripts/gen/write-verify-receipt.py` from
`python3 scripts/manager.py verify full` output.

## Owner

compiler team (diagnostics + wasm backend)

## Removal condition

Each skip is removed when the selfhost compiler implements the missing
diagnostic or compile path for that fixture. The fixture is removed from
the skip list and runs as a normal check.

## Validation command

```bash
python3 scripts/check/check-t3-wasm-validate.py
python3 scripts/manager.py selfhost diag-parity
```

## Current count

23 T3 compile skips + 26 diagnostic parity skips = 49 total skips

## New-failure ratchet

No new skips may be added. The count must only decrease. Any new skip is
a regression and blocks merge.
